Automatically prefetch all links(URLs) in the viewport and make your website page load faster like a rocket.

== Description ==

Auto Prefetch URL in Viewport is an ultra-lightweight javascript plugin (<1 KB) that will automatically prefetch all links(URLs) in the viewport and make your website page load faster like a rocket. It will be supported and maintained until at least 2022, or as long as is necessary.

Once activated, it will embed a small javascript file(auto-prefetch-url-in-viewport.min.js) on your website. There is no other configuration, the auto prefetch URL in viewport settings screens are enabled or disabled by either enabling or disabling this plugin.

== Frequently Asked Questions ==

= Are there any settings? =

No, there are no settings. Once activated, this plugin will automatically prefetch all links(URLs) of your website silently in the background.
